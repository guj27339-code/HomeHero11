Политика обработки персональных данных

Демонстрационный документ для учебного MVP HomeHero.

1. Какие данные собираются

Имя, email, телефон, город, роль пользователя, аватарка, сведения о заказах, отзывы, обращения в поддержку, баланс, а также паспортные данные для верификации личности.

2. Цели обработки

Создание аккаунта, проверка личности, предотвращение мошенничества, подбор мастеров, оформление заказов, связь по заявкам, поддержка, безопасная сделка, рейтинг и улучшение сервиса.

3. Отдельное согласие

Паспортные данные относятся к чувствительному блоку проверки личности. В интерфейсе для них сделана отдельная галочка согласия на странице верификации.

4. Хранение

В учебной версии данные сохраняются в локальной SQLite-базе на устройстве, где запущен сайт. В production нужно использовать защищённый сервер, HTTPS, контроль доступа, резервное копирование и журналирование действий администраторов.

5. Права пользователя

Пользователь может запросить изменение, удаление или ограничение обработки данных через поддержку сервиса.

6. Безопасность

Пароли хранятся в виде хэша, а не открытым текстом. Аватарки сохраняются в отдельной папке загрузок. Для реального запуска нужно добавить шифрование чувствительных данных, подтверждение телефона/email и настоящую KYC-проверку.

Принципы HomeHero

  • Минимизация лишних данных
  • Отдельное согласие на паспорт
  • Пароли не хранятся открыто
  • Доступ к данным ограничен ролью
  • История заказа нужна для безопасности
