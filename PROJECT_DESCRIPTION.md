# Описание проекта HomeHero

## Короткое описание

HomeHero — MVP marketplace-платформы для поиска проверенных мастеров по домашним услугам: ремонт, уборка, сантехника, электрика, сборка мебели и другие бытовые задачи.

## Полное описание

HomeHero — это учебный MVP веб-платформы, которая соединяет клиентов и проверенных мастеров для выполнения бытовых услуг. Пользователь может зарегистрироваться, пройти верификацию, создать заказ, выбрать категорию услуги, назначить мастера, обсудить детали в чате, загрузить фото «до / после» и получить электронную гарантию после завершения работы.

В проекте реализованы роли клиента, мастера, поддержки и администратора. Для безопасности добавлены верификация личности, безопасная сделка, жалобы, тревожная кнопка, поддержка по заказам и административная панель. Для бизнес-логики предусмотрены промокоды, комиссия платформы, баланс мастера, подписка Pro, рейтинг надёжности, портфолио и аналитика.

Проект демонстрирует устройство marketplace-сервиса и может использоваться как учебная работа, портфолио-проект или основа для дальнейшей разработки.

## Дополнительные бизнес-функции

В новой версии добавлены демо-оплата, история платежей, фото проблемы при создании заказа, умная анкета по категориям, карта районов обслуживания, система споров и CRM-панель администратора. Это делает проект ближе к реальному marketplace-сервису и повышает ценность проекта для презентации или продажи.
