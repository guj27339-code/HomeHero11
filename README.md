# HomeHero

**HomeHero** — учебный MVP marketplace-платформы для домашних услуг: ремонт, уборка, электрика, сантехника, сборка мебели и другие бытовые задачи.

Проект сделан на **Python Flask + SQLite** и показывает, как может работать сервис уровня «клиент → заказ → мастер → безопасная сделка → поддержка → админка».

## Описание проекта

**HomeHero** — это MVP веб-платформы для поиска проверенных мастеров по бытовым услугам. Сервис помогает клиенту быстро создать заявку на ремонт, уборку, электрику, сантехнику или сборку мебели, выбрать мастера, обсудить детали в чате и получить гарантию после выполнения работы.

Главная идея проекта — сделать заказ домашних услуг более безопасным и прозрачным. В HomeHero есть верификация пользователей, безопасная сделка, система жалоб, поддержка, рейтинг мастеров, фото «до / после», электронная гарантия и административная панель для контроля качества.

Проект разработан как учебный MVP и демонстрирует ключевую логику marketplace-сервиса: регистрацию пользователей, разные роли, создание заказов, работу мастеров, поддержку клиентов, админку, аналитику и базовую бизнес-модель с комиссией платформы.


> Важно: это учебный проект. Для реального запуска нужны настоящие платежи, защищённое хранение документов, KYC/верификация, юридические документы и аудит безопасности.

## Production-запуск

Для сервера используйте:

```bash
gunicorn wsgi:app
```

Файл `wsgi.py` вызывает `init_db()`, поэтому база SQLite создаётся автоматически при первом запуске.

Для настоящего production лучше использовать:

- PostgreSQL вместо SQLite;
- переменную окружения `SECRET_KEY`;
- HTTPS;
- отдельное защищённое хранилище для документов;
- настоящую платёжную систему;
- аудит безопасности.
